# Budget Request Management - Frontend

This is the frontend application for the Budget Request Management System built with Next.js 15.

## Tech Stack

- Next.js 15.3
- React 19
- TypeScript
- Tailwind CSS
- React Hook Form
- Chart.js
- Axios

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

The application will run on [http://localhost:3001](http://localhost:3001).

### Build

```bash
pnpm build
```

### Production

```bash
pnpm start
```

## Environment Variables

Copy `.env.example` to `.env` and configure:

```env
NEXT_PUBLIC_API_URL=http://localhost:4005/api
PORT=3001
```

## Deployment

This frontend is configured for deployment on Vercel.
